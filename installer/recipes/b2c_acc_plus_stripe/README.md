# b2c_acc_plus_stripe

Validated SAP Commerce 2211 JDK21 recipe for the Stripe connector public
release. It enables B2C accelerator, OCC, webhook services, OPF services used by
the connector, and the Stripe extension chain.

`stripespartacussampledata` is not enabled by default. Use the
`js-storefront/stripe-spartacus-connector` package for SAP Composable
Storefront integration.
